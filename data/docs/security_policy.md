# Information Security Policy

## Password Requirements
Employees must use a unique password for all company systems and enable multi-factor authentication where available.
Passwords must not be shared through email, chat, or written notes stored in visible work areas.

## Device Security
Company laptops must have disk encryption enabled and lock automatically after 10 minutes of inactivity.
Employees must install security updates within 7 days of release unless an exception is approved by IT.

## Incident Reporting
Suspected phishing, malware, or unauthorized access must be reported to the security team within 1 hour of discovery.
Employees should include screenshots, message headers, or any other relevant context when filing an incident report.
